Proyectos

Proyecto 1

Sistema de Verificación de Tickets con QR Desarrollé un sistema para gestionar y verificar el estado de tickets mediante códigos QR. El sistema permite a los usuarios escanear el código con su dispositivo móvil para consultar información en tiempo real desde una base de datos central. Características principales: Diseño responsivo con Bootstrap. Generación y escaneo de códigos QR. Conexión a una base de datos para la actualización de estados. Tecnologías utilizadas: HTML, CSS, JavaScript, PHP, MySQL, API QR Code.

Proyecto 2

Aplicación de Calculadora para Android Creé una aplicación móvil básica para realizar operaciones matemáticas estándar. Este proyecto está orientado a aprender y aplicar conceptos de desarrollo de aplicaciones móviles. Características principales: Interfaz intuitiva y fácil de usar. Operaciones básicas: suma, resta, multiplicación y división. Implementación de validaciones para evitar errores en las entradas. Tecnologías utilizadas: Java, Android Studio.

Proyecto 3

Portal Web de Portafolio Personal. Diseñé y desarrollé un sitio web personal para mostrar mi experiencia, habilidades y proyectos. El objetivo fue crear una plataforma profesional para compartir mi trabajo con clientes y empleadores potenciales. Características principales: Navegación dinámica con Bootstrap. Secciones de biografía, especialidades, proyectos y contacto. Integración con redes sociales y formulario de contacto funcional. Tecnologías utilizadas: HTML, CSS, Bootstrap, JavaScript, PHP.
